# Keyboard Shortcuts

## Navigation

| Shortcut | Action |
|----------|--------|
| Cmd+1 | Notes view |
| Cmd+2 | Board view |
| Cmd+3 | Graph view |
| Cmd+, | Settings |
| Cmd+P | Command palette |
| Cmd+W | Close current tab |
| Cmd+\ | Toggle sidebar |

## Creating

| Shortcut | Action |
|----------|--------|
| Cmd+N | New note |
| Cmd+D | Today's daily note |
| Cmd+Shift+D | New drawing (Excalidraw) |
| Cmd+Shift+N | New board |

## Editing

| Shortcut | Action |
|----------|--------|
| Cmd+S | Save current note |
| Cmd+E | Toggle source/live mode |
| Cmd+B | Bold selected text |
| Cmd+I | Italic selected text |
| Cmd+K | Insert link |
| Cmd+Shift+R | Rename current note |
| Cmd+Shift+A | Toggle agent panel |

## Tabs

| Shortcut | Action |
|----------|--------|
| Double-click tab | Rename note |

## Sync

| Shortcut | Action |
|----------|--------|
| Cmd+Shift+S | Sync now |
