# Daily Notes

Press **Cmd+D** to open today's daily note. If it doesn't exist yet, Codex creates it from the daily template.

## Where they live

Daily notes are stored in the `daily/` folder with date-based filenames:

```
daily/
  2026-04-18.md
  2026-04-19.md
  2026-04-20.md
```

## Templates

The daily template lives at `.codex/templates/daily.md`. Edit it to match your workflow. The `{{date}}` variable is replaced with the current date.

## Using daily notes

Daily notes work well as an entry point for the day:

- Plan what you'll work on
- Log what actually happened
- Link to the notes and projects you touched
- Reflect at the end of the day

The links you create in daily notes show up in the graph, connecting your timeline to your knowledge.
